# Padrón ANR - HC Ybytymi (Next.js + Supabase)

## 1. Instalar

```bash
npm install
```

## 2. Variables de entorno

Copiar `.env.example` a `.env.local` y completar:

```bash

NEXT_PUBLIC_SUPABASE_ANON_KEY=TU_CLAVE_PUBLICABLE
NEXT_PUBLIC_WHATSAPP_NUMBER=595971689613
```

## 3. SQL en Supabase

```sql
create extension if not exists "uuid-ossp";

create table if not exists usuarios (
  id uuid primary key default uuid_generate_v4(),
  email text unique not null,
  rol text not null default 'operador',
  creado_en timestamp default now()
);

create table if not exists padron (
  id bigint generated by default as identity primary key,
  cedula text,
  nombres text,
  apellidos text,
  distrito text,
  departamento text,
  mesa text,
  local text,
  seccion text
);

insert into usuarios (email, rol)
values ('lc3storeonline@gmail.com', 'admin_general')
on conflict (email) do update set rol = 'admin_general';
```

## 4. Crear usuario en Auth

En Supabase Authentication > Users, crear:

- lc3storeonline@gmail.com

## 5. Ejecutar local

```bash
npm run dev
```

## 6. Deploy en Vercel

- Subir esta carpeta a GitHub o importarla en Vercel
- Configurar variables de entorno
- Deploy
